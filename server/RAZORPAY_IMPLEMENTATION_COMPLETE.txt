## 🚀 Quick Setup (5 Minutes)

### 1. Install Razorpay Package
```bash
cd server
npm install razorpay
```

### 2. Configure Environment
Create/update `.env`:
```env
RAZORPAY_KEY_ID=your_key_id
RAZORPAY_KEY_SECRET=your_key_secret
```

### 3. Run Database Migration
```bash
mysql -u root -p menutha_db < migrations/razorpay_integration.sql
```

### 4. Restart Server
```bash
npm start
```

## 💡 Key Implementation Highlights

### Payment Flow
```
1. Customer initiates payment
   ↓
2. Create order (POST /api/razorpay/create-order)
   - Check subscription status
   - Calculate commission
   ↓
3. Open Razorpay checkout
   ↓
4. Verify payment (POST /api/razorpay/verify-payment)
   - Verify signature
   - Update transaction
   ↓
5. Order confirmed with commission tracked
```

### Commission Calculation
```
At transaction creation:
├─ Query subscription table
├─ If subscription is ACTIVE & NOT EXPIRED
│  └─ commission = 0, status = 'none'
└─ Else
   └─ commission = amount × 2.5%, status = 'pending'
```
